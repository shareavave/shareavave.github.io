一篇去除小米电视广告的通用教程

发表于 2022-01-20 更新于 2022-02-25

简单记录一下过年在家精简家里的小米电视的思路，毕竟我是个外行，所以这个操作应该很适合像我一样的小白用户。
我希望达到的效果是删除开机广告、去除主页上所有的付费电影和综艺节目推广、删掉锁屏壁纸和不需要的视频软件、删掉所有需要开会员的功能和内容。

而作为一个电视，我又希望它能简单的观看所有的电视频道、能看我想看的电影和安装我所需要的APP、它还应该能不受限制且没有广告的投屏而且让所有系统功能和接口能正常使用。
所以我的想法是这样：删掉小米系统自带的桌面，删掉所有视频软件和广告软件以及广告插件，保留系统自己的投屏和跨屏功能并删掉投屏软件然后用哔哩哔哩投屏代替(因为我反正都要装哔哩哔哩)，然后安装一个当贝播放器播放电脑和WebDAV里的电影，用emby for TV观看热门和并不稀缺的电影和电视剧，安装一个并夕夕买的两块钱学习版直播软件收电视台节目。嗯，这就是一个电视应该有的所有功能。

（本来应该有许多截图的，但是我还不会用图床功能，所以等之后再补截图。）

开篇提要

用的是很简单的安卓adb命令行操作，因为许多像我这样的普通用户可能不会安装adb之类的工具，一边看教程一边找文件和驱动也比较繁琐，而且配置环境和变量之类的操作实在是很复杂。于是我们去看了下教程，然后做成了附件的压缩包，其他人只要下载解压然后打开解压目录就能用了。

理论上常见的小米电视和红米电视都能用，需要安卓5.0以上的安卓版本，比如小米电视4全系列、红米电视A系列、MAX系列和X系列都符合要求，小米盒子应该也可以。

思路是这样的：
当然要删掉小米电视自带的桌面，推荐太多了，眼花缭乱，没有重点，不仅长辈找不到自己想看的东西，我们自己也找不到，唾弃，并删掉。
删掉桌面之后安装一个简单明了的桌面APP，并把它设置为默认。
这个新桌面应该要足够简单，操作逻辑要符合思维惯性，学习成本要尽可能的低，并且要有一定程度上的自定义性，功能也不能有短板，最重要的是，颜值要足够高。几番筛选，Emotn UI是唯一符合要求的桌面。

删掉桌面之后顺便删掉不必要的播放器和组件，我自己是不看，但是如果你喜欢看优酷爱奇艺之类的也可以自己安装一个，安装好了之后和手机一样在电视上点击对应的图标启动即可，它们不会在主页上推荐内容，也没必要在不用它们的时候偷偷启动。

要有可恢复性：如果没操作好或者以后需要送修之类的情况，能很方便的恢复到小米系统本身的样子，而不需要繁琐的刷机操作。
如果你不喜欢删减之后的结果，只需要在“设置-关于-还原”中还原电视系统的设置并重启电视即可。

准备工作

1.让电视和操作的电脑处于同一个路由器的网络下，不能跨路由器或连接桥接路由器和信号放大器。不限制连接方式，WiFi或者有线网络都可以，但是避免出错，尽量别同时使用全局代理。

2.确保电视的Android版本是5.0或者以上，这个很重要，因为Emotn UI只能安装在5.0+的系统上，如果删除了旧的桌面而Emotn UI却没安装成功，会导致不能顺利开机。

3.将电视的MIUI TV版本升级到最新版，打开“小米电视设置”，点击“关于”，然后点一下“MIUI TV版本”看是否有更新，顺便把小面的“智能更新”设置为“关闭”。

4.回到“设置”的主页，点击第一个看着像WiFi的图标，点进去之后点击“无线网络”，点击你所连接的WiFi，然后记录下WiFi中的“IP地址”一栏的地址，它们通常是192.168.x.xxx这样的结构，记住它。

开始操作

1.打开开发者模式：

打开电视的开发者模式，按遥控器上的房子键回到主页，然后选择电视的“小米电视设置-关于”，光标移动到“设备型号 MITV”上，然后连续按6次遥控的OK键，屏幕上会出现提示：“您已处于开发者模式……”。
回到“设置”的首页，选择“账号与安全”，在“安全”分支中，将“安装未知来源的应用”设置为“允许”，然后将“ADB调试”也设置为“允许”。

2.连接电视：

在电脑上解压附件中的压缩文件，注意是右键点击文件-解压到…，而不能直接双击打开它。

打开解压之后的文件夹，它的文件结构应该是这样的：

说明.txt
双击运行这个.bat
EmotnUI.apk
fastboot.exe
AdbWinUsbApi.dll
AdbWinApi.dll
adb.exe

双击.bat文件打开，窗口上会提示你按一下回车，然后输入刚才记下来的电视IP地址，回车。

此时将目光转向电视机！
电视会弹出一个选择 “是/否”或“允许/拒绝”的弹窗，此时务必选择“是”或者“允许”。
（误操作了也没关系，脚本会一直重复尝试连接，所以电视过会儿会再次弹出来，你也可以关闭窗口重新打开。

然后就等脚本跑完，最后显示“好了，完成了。”，就说明成功了。

此时你会看到电视上的页面变了，表示桌面已经安装完成。

3.其它操作

其实直到上面为止就操作好了，但是你可能想安装诸如腾讯视频、爱奇艺、哔哩哔哩、优酷、快手、抖音等别的应用程序，此时有三个选择：

1.直接去对应软件的官网下载安装包，然后拷贝到U盘里，把U盘插上电视，手动安装。

2.安装一个当贝助手，然后去当贝助手里安装你想要的软件，这也是可行的。

3.比较推荐的操作是将你想要安装的软件下载到电脑里，然后在刚才解压的XiaomiTV-litle这个文件夹中按住shift键并在空白的地方单击鼠标左键，在弹出的选项框中选择“在Windows终端 中打开”。然后输入adb conncet 你的电视IP，回车。

屏幕上会提示：able to connected to 192.168.x.xxx ，表示连接上了。

此时输入adb install空格，然后用鼠标将你要安装的apk文件拖到这排字的后面，它会变成一个路径，然后按回车，系统就会开始远程安装。

此时它会显示一个速度的文本，类似于这样：

xxx x file pushed. x MB/s (xxx bytes in x.xxx s)pkg: /data/local/tmp/xxx.apk

最后会显示一个“Success”，表示安装完成，此时可以输入adb install空格，拖入下一个要安装的apk重复操作。

（注意：所有需要的工具都在这个文件夹里，所以只能在这个文件夹中开始操作。
说得比较复杂，其实就是输入adb install 然后把安装包拖进窗口里然后回车，简单的两步。

题外话

这个脚本本身是当贝的官方或水军写的，当贝似乎在想方设法扩张自己的市场占有，居然为每一个电视和电视盒子写了“将系统桌面替换成当贝桌面”的教程。
但我个人感觉当贝桌面并不好用，完成度虽说足够高，但颜值算不上精雕细琢，而且许多功能有会员限制，桌面本身有广告和推广，推广的节目也需要开会员或付费。当贝桌面的版本错综复杂，强制升级、功能付费、关注微信等槽点也很多。但如果你确实喜欢当贝桌面，去当贝官网下载桌面替换掉EmotnUI.apk，然后再执行脚本即可。
当贝这些年为电视和机顶盒行业做的改变我是一路看过来的，虽说由于本身的市场属性，推广和盈利目的操作十分明显，但是当贝确实很大程度上改变了国内电视应用的氛围和环境，而且当贝播放器和当贝应用商店确实功能全面，我不吝表扬。

所以我还是把他当贝桌面的文件给替换掉了，觉得Emotn UI符合要求些，这个脚本所用到的所有功能都是adb官方文档中的功能，也不算是侵权吧。
而且当贝写的精简列表太过极端，删减掉了许多系统互相关联的功能和硬件接口调用的功能，这导致电视的设置页面读不到对应的信息，从而变得卡顿和闪退。
相比之下本文中的文件的删减则温和许多，保证了设置页面的流程操作，所有按钮都可以点击，并且依然能识别HDMI和USB的接口变动。
